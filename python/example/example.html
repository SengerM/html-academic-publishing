Introduction

The HTML format is really nice, but writing in pure HTML is a pain. The same happens with Latex. That is why things like Lyxhttps://www.lyx.org/. exist. Writing complex math equations like \[ f\PARENTHESES x=\PARENTHESES{1-\xi}\cos\theta+\intop_{0}^{\infty}\frac{\DIFERENTIAL k}{2\pi}\frac{\sqrt{k-\Xi}}{\omega+\varepsilon} \] is really easy in LyX. It is also much more visual friendly.
I like LyX so much for this kind of stuff that I developed this script to automate the task. It supports many of the most common things like footnotesThis is an example footnote with inline math $x^{3}$ and displayed math \[ \PARENTHESES{i\cancel{\partial}-m}\psi=0\text{,} \] a reference to the bibliography    and also a reference to ., math, cross-references (see  and ), listings, etc.
