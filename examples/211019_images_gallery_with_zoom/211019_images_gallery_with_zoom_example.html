To use the images_gallery_with_zoom_211019 just include the CSS file by adding <link rel="stylesheet" href="https://sengerm.github.io/html-academic-publishing/css_and_scripts/images_gallery/images_gallery_with_zoom_211019.css"> in the head of your HTML. Then wrap all the images (or videos or whatever) you want to display as a gallery with miniatures and zoom on click within a div with class="images_gallery_with_zoom_211019". Result is below. Not perfect yet.

This photo gallery is useful for presentations, in which you want to show many pictures and have the possibility to show some of them big if required.

In order for the zoom to be held after you release the mouse button, what works for me now is to drag a little bit the mouse before releasing the button. It's a hack that works.

It is possible to set the height of miniatures for each gallery with the argument --miniatures_height. See examples in and .
